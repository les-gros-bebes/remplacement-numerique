# Guide Fit-Coach AI

Ce document présente le contenu généré pour l'assistant sportif Fit-Coach. Il est conçu pour être intégré dans une interface web.

## 1. Questionnaire de Profilage (QCM)

Le but est de cerner rapidement le profil de l'utilisateur en 8 questions.

1.  **Objectif** : Perte de poids / Muscle / Endurance / Souplesse
2.  **Niveau** : Débutant / Intermédiaire / Avancé
3.  **Disponibilité** : <20min / 20-45min / >45min
4.  **Lieu** : Maison / Extérieur / Salle
5.  **Matériel** : Aucun / Basique / Complet
6.  **Limitations** : Aucune / Dos / Genoux / Épaules
7.  **Type d'effort** : HIIT / Cardio / Force
8.  **Motivation** : Visuel / Ressenti / Challenge / Mental

## 2. Logique de Profilage

L'algorithme combine principalement l'**Objectif** et le **Niveau** pour définir le programme de base.

- _Exemple_ : "Perte de poids" + "Débutant" = Programme cardio doux et renforcement global.
- Les contraintes (Matériel, Lieu, Blessures) agissent comme des filtres pour exclure ou adapter certains exercices.

## 3. Exemple de Profil : "Remise en forme Tonique"

> **Description** : Un programme accessible pour brûler des calories et tonifier l'ensemble du corps sans impact violent.
> **Cible** : Débutant, Objectif Perte de poids, Matériel basique.

## 4. Exercices Sélectionnés

Voici 5 exercices clés adaptés à ce profil, avec illustrations simplifiées.

### A. Squat au poids du corps

_Cibles : Cuisses, Fessiers_

```
   O
  /|\
  / \

   O
  /|\
  _ _
```

**Instructions :**

- **Posture** : Pieds largeur d'épaules, dos droit.
- **Mouvement** : Descendez les fesses en arrière comme pour vous asseoir.
- **Erreur à éviter** : Genoux qui rentrent vers l'intérieur.
- **Adaptation** : Descendre moins bas si débutant.

---

### B. Pompes (Adaptées)

_Cibles : Pectoraux, Bras, Gainage_

```
      __O
    /  _\
   /  /  \

   __O
  /_ _\
 / /  \
```

**Instructions :**

- **Posture** : Mains au sol, corps aligné. Sur les genoux pour débuter.
- **Mouvement** : Amenez la poitrine au sol puis repoussez.
- **Erreur à éviter** : Dos cambré.

---

### C. Gainage Planche

_Cibles : Abdominaux profonds_

```
     O
   //_\\
  //   \\
```

**Instructions :**

- **Posture** : Sur les avant-bras, corps rectiligne.
- **Mouvement** : Tenir la position immobile.
- **Erreur à éviter** : Fesses trop hautes ou dos creusé.

---

### D. Fentes Alternées

_Cibles : Jambes, Équilibre_

```
   O      O
  /|\    /|\
  / \    _|_
```

**Instructions :**

- **Posture** : Debout, grand pas en avant.
- **Mouvement** : Fléchir les deux jambes à 90°.
- **Erreur à éviter** : Genou avant qui dépasse la pointe du pied.

---

### E. Mountain Climber

_Cibles : Cardio, Abdos_

```
    __O
   / /\
  /_/  \
```

**Instructions :**

- **Posture** : En planche haute (bras tendus).
- **Mouvement** : Ramener alternativement les genoux vers la poitrine avec rythme.
- **Erreur à éviter** : Bassin qui monte et descend trop.

## 5. Recommandations Produits Decathlon

Pour accompagner ce profil, voici une sélection d'équipements essentiels :

1.  **Tapis de sol Fitness 10mm** : La base pour le confort.
2.  **Élastiques de résistance** : Pour intensifier les squats et fentes sans haltères.
3.  **Kit Haltères 10kg** : Évolutif pour le renforcement des bras.
4.  **Kettlebell 6kg** : Pour varier les plaisirs (Swing, Goblet Squat).
5.  **Rouleau de massage** : Pour la récupération après séance.
